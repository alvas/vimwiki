## REST

Representational State Transfer

## REST Constraints

The idea of a RESTfull system is based on the conecpts of constraints that limit how a distributed system can be constructed.

* Client/Server
* Stateless
* Caching
* Uniform Interface
* Layered System
* Code on Demand

## HTTP Status Codes

* Success Codes (2xx)
* Redirection (3xx)
* Client Errors (4xx)
* Server Errors (5xx)
